```sh
ctrl + ←/→
```
## 2. 搜索特定的进程
`ntop`是类`vi`操作, 输入`/`然后输入感兴趣的进程名即可搜索, 如 `/qq`

## 3. 杀死进程
输入`:kill 123456`即可
