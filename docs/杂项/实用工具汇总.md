## 4. win软件安装

```sh
# sqlite可视化工具
scoop install sqlitebrowser
# 类似 top
scoop install ntop
# 文件搜索工具
scoop install fzf
# 搜索工具
scoop install ag
# 替代 cat
scoop install bat
# 资源监控, 类似 ntop
scoop install btop
# 可视化追踪工具
scoop install nexttrace
# 替代 ls
scoop install eza
# PowerToys
scoop install PowerToys
# 安装 dust，一个更现代的磁盘使用分析工具（替代 Windows 的 dir / du / 文件资源管理器查看磁盘大小）
scoop install dust
# 安装 fd，一个比 Windows 自带的 `dir / find` 或 `where` 更快更友好的文件搜索工具
scoop install fd
# 安装 bottom（btm），一个类似 Linux top/htop 的系统资源监控工具（替代 Windows 任务管理器或者资源监视器）
scoop install bottom
# 安装 diff-so-fancy，让 git diff 更易读，增强 Git 自带 diff 的可读性（替代普通 git diff）
scoop install diff-so-fancy
```
