## 4. win软件安装

```sh
# 类似 top
scoop install ntop
# 文件搜索工具
scoop install fzf
# 搜索工具
scoop install ag
# 替代 cat
scoop install bat
# 资源监控, 类似 ntop
scoop install btop
# 可视化追踪工具
scoop install nexttrace
# 替代 ls
scoop install eza
```
